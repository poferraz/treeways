# Urban Canopy Engine

A map-first, accessible field guide for urban trees. Vancouver is the initial city pack.

## Quick start

```sh
npm install
npm run city:build
npm run dev
```

Run `npm run test:unit`, `npm run check`, `npm run build`, and `npm run check:bundle` before submitting changes. The engine uses GPU MapLibre layers and a worker-decoded city pack; it does not claim routing or the complete basemap works offline.

City data and media have independent licences. See each city pack’s `LICENSE-DATA.md` and attribution.
