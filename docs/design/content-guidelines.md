# Content guidelines

## Voice

Write as a precise, calm field guide. Prefer observable facts and plain directions. Avoid hype, certainty that the data does not support, and folklore presented as instruction.

Good: "Flowering now. Petals are usually most visible from late March to April."

Avoid: "A magical spring spectacle you cannot miss."

## Tree identity

- Common name is the primary heading in title case.
- Scientific name follows in italics, formatted as Genus species. Genus is capitalized, species is lowercase.
- Use a data label for cultivar or hybrid rather than folding it into a scientific name without source support.
- Category labels are factual: Edible, Blossoming, Shade, Municipal background, or Curated.
- Current seasonal state appears as a sentence with the current signal first: "Harvesting now", "Likely dormant", or "Bloom period expected soon".

## Data confidence and omissions

State what is known. If height, diameter, address, condition, source, date, media, or phenology is missing, show "Not recorded" or omit the row when it does not affect an action. Never use question marks as data values. Use "Estimated" only with a source-supported basis.

Use careful harvest language: "Reported harvest period" or "Harvest period in this dataset". The product must not imply that every fruit is safe, accessible, or permitted to pick. Where relevant, add a concise local guidance link.

## Distance, bearing, and time

- Under 1 km, present distance in metres. At 1 km or more, round to one decimal place in kilometres.
- Bearings use plain-language direction such as north-west, not unexplained degrees.
- Walking time comes from the routing provider and is labelled as an estimate.
- Last updated uses a specific date in locale-aware format, not relative time as the only representation.

## Media and attribution

Only show species media when the species, license, and attribution are verified. Store creator, source URL, license, crop, alt text, and verification status with the media record. Do not use generic stock photography as proof of a particular specimen. When trusted media is unavailable, use a labeled botanical placeholder or omit the media block.

Attribution is placed near the image in a compact details disclosure, with a direct source link. It is not decorative copy.

## Labels and actions

Use verbs that describe the outcome: "Add to route", "Navigate", "Show nearby trees", "Apply filters", "Clear filters", "Move stop earlier", and "Report an observation". Avoid vague labels such as "Explore", "Discover", or "Submit" without an object.

## Empty and error copy

| Situation | Recommended copy | Action |
| --- | --- | --- |
| No results | "No curated trees match these filters in this area." | "Clear filters" and "Search another area" |
| No nearby species match | "No nearby records for this species. Try a wider map area." | "Show matching species" |
| Offline | "You are offline. Showing data already saved on this device." | "Try again" when a request is required |
| Route failure | "Route could not be calculated." | "Try again" and "Navigate to this tree" |
| Media unavailable | "Verified image unavailable." | no action unless a source exists |
